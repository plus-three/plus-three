# plus-pro-components docs

## 本地开发命令的区别

| 命令       | 作用                             | 备注                         |
| ---------- | -------------------------------- | ---------------------------- |
| pnpm dev   | 预览的文档是在线的组件           |                              |
| pnpm debug | 预览的文档是本地最新的组件       | 写作本地最新组件时使用       |
| pnpm start | 预览的文档是本地最新打包后的组件 | 测试最新组件打包后效果时使用 |
